## Usage Examples

### Basic Usage
```python
from airbornehrs import AdaptiveFramework, AdaptiveFrameworkConfig

# Create config
config = AdaptiveFrameworkConfig(
    learning_rate=0.001,
    enable_consciousness=True
)

# Wrap your model
framework = AdaptiveFramework(your_model, config)

# Use like normal PyTorch
for X_batch, y_batch in data_loader:
    output = framework(X_batch)
    loss = criterion(output, y_batch)
    framework.optimizer.zero_grad()
    loss.backward()
    framework.optimizer.step()

# Consolidate after learning
framework.consolidate_memory(X_train)
```
